
# How to Import Dashboards in Grafana

Grafana allows you to import dashboards in **three different ways**:

1. **Upload a dashboard JSON file**  
2. **Paste a Grafana.com dashboard URL or ID**  
3. **Paste dashboard JSON text directly into the text area**

---

## Step 1: Log In to Grafana

* Open your browser and navigate to your Grafana URL.  
* Log in with your credentials.

---

## Step 2: Open the Dashboard Import Page

1. Click the **menu (☰)** on the left.
2. Go to **Dashboards → Manage**.
3. Click **Import** in the top-right corner.

---

## Step 3: Choose Your Import Method

On the Import page, choose one of three options:

---

### Option 1: Upload a Dashboard JSON File

1. Click **Upload JSON file**.  
2. Select the `.json` file from your computer.  
3. Choose the **data source** that the dashboard should use.  
4. Click **Import**.

---

### Option 2: Paste a Grafana.com Dashboard URL or ID

1. Visit: https://grafana.com/grafana/dashboards  
2. Copy the **Dashboard ID** or **full URL**.  
3. In Grafana, paste it into the **Import via Grafana.com** field.  
4. Click **Load**.  
5. Select your **data source**.  
6. Click **Import**.

---

### Option 3: Paste Dashboard JSON Text Directly

1. Copy the full dashboard JSON.  
2. In Grafana, scroll to the **Import via panel JSON** text area.  
3. Paste the JSON text.  
4. Select the **data source**.  
5. Click **Import**.

---

## Step 4: Verify the Imported Dashboard

* The dashboard will open automatically.  
* Confirm that panels load data correctly.  
* Adjust visuals if needed.

---

## Step 5: Save and Organize Your Dashboard

1. Click **Save** (top-right).  
2. Name your dashboard (e.g., *Network Overview – Imported*).  
3. Optionally move it to a folder.

---

## Key Takeaway

Grafana supports three import methods:

✔ Upload a **JSON file**  
✔ Paste a **Grafana.com dashboard ID or URL**  
✔ Paste **JSON text** directly  

This makes it easy to reuse and share dashboards.
