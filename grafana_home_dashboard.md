# How to Set Home Dashboard in Grafana

## ✅ Organization-Wide Home Dashboard
1. **Log in as an Organization Admin**.
2. Click the **gear icon (Configuration)** in the left sidebar.
3. Select **Preferences**.
4. Under **Home Dashboard**, choose the dashboard you want from the dropdown.
5. Click **Save**.

---

## ✅ Per-User Home Dashboard
1. Log in as the user (or have them do it).
2. Click the **profile icon** (top-right corner).
3. Select **Preferences**.
4. Under **Home Dashboard**, pick the desired dashboard.
5. Click **Save**.

---

### ⚠️ If You Don’t See Preferences
- You might not have **Admin rights** for the organization.
- Ask an Org Admin to set it globally or use **API/provisioning**.

---

### ✅ Optional Automation
- Use **Grafana API** or **provisioning YAML** to set defaults automatically.
