
# How to Create a New Organization in Grafana 12

## Prerequisites
- You must be signed in as a **Grafana Server Administrator**.

## Steps to Create a New Organization
1. Sign in to Grafana as a **server administrator**.
2. In the left‑side menu, click **Administration → General → Organizations**.
3. Click **+ New org**.
4. Enter the **name** of the new organization and click **Create**. Grafana will:
   - Create the organization
   - Add you as the **organization administrator**
   - Open the **Default Preferences** page
5. Optionally configure organization preferences:
   - Home dashboard
   - Time zone
   - Week start day
6. (Optional) Add users to the new organization.

## Notes
- Organizations isolate dashboards, data sources, alerts, folders, and other assets.
- Users can belong to multiple organizations.
