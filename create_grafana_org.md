
# How to Create a New Organization in Grafana 12

## Prerequisites
- You must be signed in as a **Grafana Server Administrator**. citeturn1search1

## Steps to Create a New Organization
1. Sign in to Grafana as a **server administrator**. citeturn1search1
2. In the left‑side menu, click **Administration → General → Organizations**. citeturn1search1
3. Click **+ New org**. citeturn1search1
4. Enter the **name** of the new organization and click **Create**. Grafana will automatically:
   - Create the organization
   - Add you as the **organization administrator**
   - Open the **Default Preferences** page citeturn1search1
5. Optionally configure organization preferences:
   - Home dashboard
   - Time zone
   - Week start day citeturn1search1
6. Add users to the new organization (optional). For details, see Grafana’s user management documentation. citeturn1search5

## Notes
- Organizations isolate dashboards, data sources, alerts, folders, and other assets. citeturn1search1
- Users can belong to multiple organizations.
