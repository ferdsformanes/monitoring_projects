# Export Hosts from Centreon Web UI to CSV

1. **Log in to Centreon Web UI**  
   Go to `http://<Centreon-IP>/centreon` and log in as admin.

2. **Navigate to Host List**  
   - Click **Configuration â†’ Hosts**.

3. **Export to CSV**  
   - At the top-right of the host list, click **Export**.  
   - Choose **CSV** format.  
   - Save the file to your computer.
